# 引脚定义

## GPIO 电压

| GPIO       | 电压 | 最高  |
| ---------- | ---- | ----- |
| 所有的GPIO | 3.3V | 3.63V |
| SARADC_IN4 | 3.3V | 1.98V |

## GPIO 接口

Radxa NX5 IO 提供了一个40pin针脚的GPIO座子，兼容于市面上大部分传感器的应用。

**_提示:_ 实际兼容情况以使用为准。**

- Radxa NX5 IO 有一个40针扩展接头。每个引脚都用颜色区分。
